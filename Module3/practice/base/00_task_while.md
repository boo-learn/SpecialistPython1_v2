## "Обратный отсчет"

### Задание

Выведите все целые числа от n до 0 включительно.

### Формат входных данных

Дано целое положительно число n.

### Формат выходных данных

Выведите все числа, требуемые по условию задачи.

### Решение задачи

```python
# TODO: you code here...
number = int(input())

i = 0
while number >= i:
    print(i)
    i += 1

```
