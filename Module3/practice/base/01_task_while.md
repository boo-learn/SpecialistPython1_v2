## "Я учу питон"

### Задание

Выведите фразу "Я учу питон" n раз

### Формат входных данных

Дано целое положительное число n.

### Формат выходных данных

Вывести текст "Я учу питон" заданное количество раз.

### Решение задачи

```python
number = int(input("Введите число: "))
count = 0
while count < number:
    print("Я учу питон")
    count += 1
```
