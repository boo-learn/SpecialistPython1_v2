## "Цифры числа-3"

### Задание

Дано трехзначное число. Найдите его первую и последнюю цифру.

**Примечание**: решите задачу без преобразования числа к строке

### Формат входных данных

Дано целое положительное трехзначное число.

### Формат выходных данных

Требуется вывести первую и последнюю цифру числа.

### Решение задачи

num3dig = int(input("number 3 digits: "))

first_dig = num3dig//100
second_dig = (num3dig - first_dig * 100)//10
third_dig = num3dig - first_dig * 100 - second_dig * 10

print("first digit: ", first_dig)
print("third digit: ", third_dig)


### Данные для самопроверки
|  number  | |   first   |  last    |
| :---: | ---| :---: | :---: | 
|   143   | |   1  |   3  |
|   894   | |   8  |   4  |
|   666  |  |   6  |   6  |
|   240  |  |   2  |   0  |
