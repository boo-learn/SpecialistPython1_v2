## "Удалить запятые"

### Задание

Удалите из строки все запятые.

### Формат входных данных

Дана произвольная строка.

### Формат выходных данных

Вывести исходную строку без запятых.

### Решение задачи

str = input("Введите произвольный текст: ")
del_str = str.replace( ',', '')
print(del_str)
