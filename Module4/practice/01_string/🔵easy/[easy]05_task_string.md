## "Какая длиннее"

### Задание

Даны две строки. Выведите ту, в которой больше символов. \
Если символов одинаково, выведите любую.

### Формат входных данных

Даны две произвольные строки.

### Формат выходных данных

Вывести наиболее длинную или любую, если длина строк одинаковая.

### Решение задачи

```python
string_1 = input("Введите текст_1: ")
string_2 = input("Введите текст_2: ")
S1 = len(string_1)
S2 = len(string_2)
if S1 > S2:
    print(string_1)
elif S1 < S2:
    print(string_2)
else: print(string_1 or string_2)
```
