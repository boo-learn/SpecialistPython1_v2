## "Подсчет символов"

### Задание

Подсчитать количество букв «б» во введенной строке

### Формат входных данных

Дана произвольная строка.

### Формат выходных данных

Вывести количество найденных букв.

### Решение задачи

```python
text = input("Введите текст: ")
mssg = text.count("б")
print(mssg)
