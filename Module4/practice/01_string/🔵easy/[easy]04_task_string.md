## "Подсчет символов"

### Задание

Подсчитать количество букв «б» во введенной строке

### Формат входных данных

Дана произвольная строка.

### Формат выходных данных

Вывести количество найденных букв.

### Решение задачи

```text = str(input('Введите текст:'))
print(text.count("б"))
```
