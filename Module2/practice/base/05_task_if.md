## "FooBar"

### Задание

Дано целое число. \
Если оно делится на 3 без остатка - вывести "Foo", \
если делится на 5 - вывести "Bar", \
а если делится на 3 и на 5 - вывести "Foobar". \
Для всех остальных случаев не выводить ничего.

### Формат входных данных

Дано целое число.

### Формат выходных данных

Вывести "Foo", "Bar" или "Foobar"

### Решение задачи

```python
number = int(input("целое число:"))
output_string = ""
if number % 3 == 0:
    output_string = output_string + "Foo"
if number % 5 == 0:
    output_string = output_string + "Bar"
print(output_string)

```
