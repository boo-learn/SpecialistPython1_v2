## "Трехзначное ли"

### Задание

Проверьте, является ли данное число трехзначным.

### Формат входных данных

Дано целое число.

### Формат выходных данных

Вывести "Да", если данное число трехзначное и "Нет" в противоположном случае.

### Решение задачи

```python
n = int(input())
if 100 < n <= 999:
    print("YES")
else: print ("NO")
```
